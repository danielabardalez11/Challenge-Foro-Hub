# Challenge-Foro-Hub
Foro Hub API

Descripción:
API REST en Java con Spring Boot que simula un foro académico. Los estudiantes pueden crear tópicos con dudas y los profesores o moderadores pueden responderlos. El proyecto practica CRUD, autenticación con JWT y persistencia de datos con MySQL.

# Funcionalidades:

Registro de usuarios y asignación de roles (estudiante, profesor, moderador).

Inicio de sesión con JWT y protección de rutas.

Gestión de tópicos: crear, listar, consultar, actualizar y desactivar.

# Tecnologías:

Java 25

Spring Boot

Spring Security

JWT

Hibernate / JPA

MySQL

Lombok

Maven

# Base de datos:

Usuarios: información personal y rol.

Registro: credenciales de login.

Tópicos: título, mensaje, autor, fecha y estado.


# Objetivo:
Practicar desarrollo de APIs RESTful con Spring Boot y conexión a MySQL.
